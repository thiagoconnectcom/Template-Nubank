# Template-Nubank-React-Native

para entender como funciona a parte da instalação acesse o link da rockeseat 

-https://docs.rocketseat.dev/ambiente-react-native/android/windows

para emular o projeto usando o android via Wifi acesse 

-https://docs.rocketseat.dev/ambiente-react-native/wifi/android

para emular o projeto usando o ios via Wifi acesse 

https://docs.rocketseat.dev/ambiente-react-native/wifi/ios

para emular o projeto usando o android via USB acesse 

-https://docs.rocketseat.dev/ambiente-react-native/usb/android

para emular o projeto usando o ios via USB acesse 

https://docs.rocketseat.dev/ambiente-react-native/usb/ios

Baixer o projeto e digite 
-$npm install

depois abra o cmd como root e digite 
-adb
-adb devices

se estiver com seu celular conectado aparecerar nessa tela e ao lado estara escrito DEVICES,se estiver tudo certo só rodar o comando no GitBash

-$react-native run-android ou $react-native-run-ios

e Pronto!
